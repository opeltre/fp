Installation
============

Note: this project is still under development.

Unstable
---------

If you have pip installed::
    
    git clone https://github.com/opeltre/fp
    cd fp && pip install -e .

To use `fp` within a poetry environment::

    git clone htts://github.com/opeltre/fp
    cd fp && poetry install
